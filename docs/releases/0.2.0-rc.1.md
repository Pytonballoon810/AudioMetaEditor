Date: 2026-03-24
Type: minor
Status: build-validated (awaiting manual smoke QA)

## Scope summary

This release candidate packages the production-readiness refactor stream:

- typed IPC contracts + runtime payload validation
- renderer feature-hook split and bridge centralization
- WaveSurfer lifecycle hardening
- file I/O hardening for large audio loads
- CI quality gates and strict TypeScript settings
- architecture, performance, security, and release documentation
